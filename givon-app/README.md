# Givon Defense — פורטל מודיעין אסטרטגי

כלי עבודה פנימי לזיהוי הזדמנויות, שותפים, משקיעים ותנועות מתחרים.

## פריסה ב-Vercel (10 דקות)

### שלב 1 — העלאה ל-GitHub
1. היכנס ל-github.com → New Repository
2. שם: `givon-intel-portal` (Private ✓)
3. העלה את כל התיקייה הזו

### שלב 2 — חיבור ל-Vercel
1. היכנס ל-vercel.com עם חשבון GitHub
2. לחץ "Add New Project"
3. בחר את ה-repo `givon-intel-portal`
4. Framework Preset: **Create React App**
5. לחץ Deploy

### שלב 3 — קבלת URL
תוך ~2 דקות מקבלים URL כמו:
`https://givon-intel-portal.vercel.app`

אפשר לשנות ל-custom domain בהגדרות Vercel.

## הרצה מקומית
```
npm install
npm start
```
